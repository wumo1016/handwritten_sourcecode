## 相关包
> `rollup` 打包工具  
> `@babel/core` `@babel/preset-env` 将ES6等转换为ES5  
> `rollup-plugin-babel` 将rollup与babel关联起来  
> `concurrently` 同时执行两个 node 命令  
> `koa-static` 访问静态文件  
> `typescript` 支持ts  
> `typescript` typescript  
> `@rollup/plugin-node-resolve` 解决依赖第三方模块  
> `@rollup/plugin-json` 支持引入json  
> `execa` 使用js执行node命令
